# hexa.test
 Test Junior на Yii Framework 2

 1. Склонировать репозиторий.
      ```
     git clone "https://github.com/gitganzyuk/hexa.test"
     ```
 2. Выполнить обновление зависимостей
     ```
     composer update
     ```
 3. Настроить подключение к БД в конфиге:
    ```
    /config/db.php
    ```
 4. Запустить миграции:
    ```
    php yii migrate
    ```
